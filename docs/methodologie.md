# Methodologie
Max 400 woorden.   
Beschrijf je methodologie (voor sem 1, enkel SEM1, zie les methodologie).    
   
Maak hierbij gebruik van een zelf gemaakte afbeelding om je tijdlijn weer te geven. Besteed hier voldoende aandacht aan. Dit is de kapstok van je project.   

Op deze tijdlijn moeten minimaal
1. een *tijdsindicatie* te zien zijn (e.g. maanden of kwartalen),
2. moeten *fasen* te zien zijn (gekoppeld aan doelstellingen van die fase, wat je wilde bereiken)  en
3. moeten *activiteiten* te zien zijn (uitgevoerde *analyses* en *onderzoeksmethoden*, inclusief N voor de steekproefgrootte, vb. *user testing (N=5)* wanneer je een test uitvoerde bij 5 personen).

Bekijk zeker ook eens [voorbeelden van eerdere jaren](https://github.com/basbaccarne/human-centered-design) 
(voor SEM1 betreft dit uiteraard slechts de helft van deze visualisaties). We bouwen hier in het tweede semester op verder.   


Neem hier ook een tijdlijn in op waarin we de evoluties van de prototypes kunnen zien.
